### Install AWS-Amplify Cli
npm install -g @aws-amplify/cli

### Configure Amplify
amplify configure

### Create a React App
npx create-react-app <name of your app>

### Change Directory 
cd <name of your app>

### Initialize Amplify
amplify init

### Auth your Amplify
amplify add auth

### Push Amplify
amplify push

### Install Amplify UI React
npm install aws-amplify @aws-amplify/ui-react
